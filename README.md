# Nutricius Bot

Telegram-бот для анализа состава продуктов питания по фотографии этикетки.

## Что делает

- Принимает фото этикетки с составом
- Распознаёт ингредиенты через Vision LLM (Gemini / Claude)
- Выдаёт структурированный анализ: NOVA-класс, скрытый сахар, Е-добавки, маркетинговые манипуляции
- Справочник Е-добавок — детерминированный (без галлюцинаций AI)

## Стек

- **Python** + pyTelegramBotAPI
- **Gemini 2.5 Flash** через OpenRouter (Vision LLM)
- **Supabase** (PostgreSQL) — rate limiting и логирование
- **Railway** — деплой

## Быстрый старт

1. Клонируй репозиторий
2. Скопируй `.env.example` → `.env` и заполни токены
3. Установи зависимости: `pip install -r requirements.txt`
4. Запусти: `python bot.py`
